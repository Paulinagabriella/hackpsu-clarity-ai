# Clarity AI

Clarity AI is a real-time social media safety agent that analyzes posts for:
- Privacy risks
- Misinformation
- Harmful communication
- Mental wellbeing impact

## Features
- AI-powered post analysis
- Privacy detection using regex
- Tone and misinformation insights
- Safer rewrite suggestions

## Tech Stack
- FastAPI (backend)
- React + Vite (frontend)
- Google Gemini API (AI)
- ElevenLabs (optional voice)

## How to Run

### Backend
cd backend  
.venv\Scripts\activate 
uvicorn main:app --reload  

### Frontend
cd frontend  
npm install  
npm run dev
